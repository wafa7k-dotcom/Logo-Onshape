# Personal Logo Design

## Project Description
This project presents a simple personal logo designed using Onshape.
The logo represents my identity as an Information Technology student through a clean and modern design. After completing the sketch, the logo was converted into a 3D model using the Extrude feature.

## Software Used
- Onshape

## Design Features
- Simple and clean personal logo.
- 3D model created using Extrude.
- Suitable for 3D printing.
- Exported in STL format.
